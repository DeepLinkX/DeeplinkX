# Contributor Lanes

## Roles

## Owner
- Owns category narrative and final editorial approval.
- Approves merge readiness for strategy and docs changes.
- Maintains KPI decisions and backlog priorities.

## Contributor A (Docs/Technical)
- Owns recipe correctness and fallback behavior explanations.
- Keeps snippets aligned with current API behavior.
- Reviews technical accuracy of claims.

## Contributor B (Distribution/Strategy)
- Owns query map, references, SO targets, and article outlines.
- Maintains external-facing messaging consistency.
- Updates tracking docs and status tables.

## Working Agreement
1. No runtime/API changes in this promotion cycle.
2. Every public claim must include proof links.
3. Use small PRs by lane for faster review.
4. Cross-review at least one PR per week.
