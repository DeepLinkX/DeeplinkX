# Review SLA

## Scope
Applies to docs, strategy, and distribution planning PRs in this cycle.

## Service Levels
1. Initial review response: within 48 hours.
2. Follow-up review after updates: within 24 hours.
3. Merge decision after approvals: within 24 hours.

## Review Requirements
1. At least one reviewer from a different lane.
2. Owner approval required for README and strategy docs.
3. No merge if evidence links are missing.

## Priority Rules
1. Blockers: incorrect API snippets, broken links, unsupported claims.
2. High priority: query coverage or claim-proof gaps.
3. Normal priority: wording and editorial polish.

## Escalation
If SLA is missed for two consecutive items:
1. Re-scope task size.
2. Split into smaller PRs.
3. Reassign owner for the week.
